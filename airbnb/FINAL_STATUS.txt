🚀 CÓMO USAR
═══════════════════════════════════════════════════════════════════════════════

  1. Abre la aplicación
  
  2. Haz clic en "📁 Cargar JSON"
  
  3. Selecciona un archivo JSON:
     • sample_airbnb.json (incluido)
     • sample_properties.json (incluido)
     • Tus propios datos JSON
  
  4. Haz clic en "Abrir"
  
  5. ¡Tus datos aparecerán en la tabla!

    → Búsqueda de texto

    → Detección de palabras clave


💡 EJEMPLOS
═══════════════════════════════════════════════════════════════════════════════

  Ejemplo 1 - Simple:
  [
    {"id": 1, "name": "Casa", "price": 100, "bedrooms": 2},
    {"id": 2, "name": "Apt", "price": 150, "bedrooms": 1}
  ]

  Ejemplo 2 - Completo:
  [
    {
      "id": 1,
      "name": "Casa",
      "description": "Bonita",

      "bedrooms": 3,

    }
  ]

  {
    "id": 1,

  }

💬 SOPORTE
═══════════════════════════════════════════════════════════════════════════════

  ❌ "No se pudo cargar el archivo"
  ✓ Valida JSON en https://jsonlint.com

  ❌ "El archivo no existe"
  ✓ Verifica la ubicación y extensión .json

  ❌ "El archivo es muy grande"
  ✓ Máximo 50MB, divide el archivo

  ❌ No aparecen datos
  ✓ Asegúrate de que sea array [] u objeto {}
